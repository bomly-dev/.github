# Support

Use the channel that best fits the request:

- Usage questions and design discussions: open a discussion in the relevant
  repository when discussions are enabled.
- Bugs and feature requests: open an issue in the affected repository.
- Security vulnerabilities: use the affected repository's Security tab for
  private vulnerability reporting, or email <contact@bomly.dev>.
- Maintainer contact: email <contact@bomly.dev>.

Please include the affected repository, version or commit, operating system,
command or workflow snippet, and sanitized logs when they help explain the
request.
